# 执笔写作

专业小说写作APP，无广告、无付费墙、功能全免费。

## 项目介绍

一款面向网文作者的专业写作工具，深度对标作家助手全链路能力，整合橙瓜写作、番茄写作核心优势，彻底剔除行业糟粕（广告、付费墙、社交绑架等），打造「极简界面为表，专业全能为里」的写作体验。

## 核心功能

- 📝 极简编辑器 - 沉浸式写作体验，稳定不丢字
- 📚 作品管理 - 卡片式作品列表，分类归档
- 🌲 树形大纲 - 作品→分卷→章节→情节节点
- 📦 素材库 - 人物设定、世界观、伏笔线索管理
- 💡 灵感速记 - 语音/文字快速记录灵感
- 🔒 数据安全 - 本地+云端双备份，历史版本回溯
- 📤 多平台发布 - 主流网文平台一键发布
- 📊 数据统计 - 码字统计、稿费管理
- 🎯 创作工具 - 敏感词检测、随机取名、一键排版

## 技术栈

- Kotlin
- Jetpack Compose
- MVVM + Clean Architecture
- Room 本地数据库
- Kotlin Coroutines + Flow
